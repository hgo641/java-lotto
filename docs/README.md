[☑] 구입 금액을 입력받는 기능

[☑] 발행한 로또 수량 및 번호를 출력하는 기능

[☑] 당첨 번호와 보너스 번호를 입력받는 기능

[x] 당첨 내역 계산 기능

[x] 수익률 계산 기능

## 예외 처리
### 구입 금액
* 구입 금액이 1,000원으로 나누어 떨어지지 않는 경우


### 로또 및 당첨 번호
* 6개의 숫자로 이루어져있지 않은 경우

* 번호가 1부터 45사이의 숫자가 아닌 경우(당첨, 보너스 번호 포함)

* 중복되는 숫자로 이루어져 있는 경우

* 입력 받은 번호가 숫자가 아닐 경우
